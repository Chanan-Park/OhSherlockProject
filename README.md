# OhSherlockProject
쇼핑몰 프로젝트 - 오!셜록

Servlet/JSP를 사용하여 개발한 차(tea) 상품을 판매하는 쇼핑몰 웹애플리케이션입니다.
<hr>
<h2>개발 환경</h2>
<ul>
  <li>Java 8</li>
  <li>Oracle RDBMS</li>
  <li>Tomcat 9.0</li>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
  <li>jQuery</li>
  <li>BootStrap 4</li>
</ul>
